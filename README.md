# Student Management System

A web-based Student Management System developed using Django to manage student registration and records efficiently.

## Features
- Student registration and data management
- CRUD operations using Django ORM
- RESTful APIs using Django REST Framework
- Search functionality using Q objects
- Form validations for email and phone number
- Admin panel with restricted access for update and delete operations

## Technologies Used
- Python
- Django
- Django REST Framework
- HTML, CSS, JavaScript
- SQLite

## Tools
- VS Code
- Git & GitHub
